# Brain Tumor Detection in MRI Images

Collaborative project focused on detecting brain tumors from MRI images using exploratory data analysis, Machine Learning, and Deep Learning techniques.

The repository includes data exploration and modeling notebooks, Python scripts for inference/classification, data versioning with DVC, and a full-stack application structure with frontend/backend components.

## My Role

I participated as a collaborator in the project, contributing to the development and/or integration of components related to the data analysis, modeling, and deployment workflow.

My main contributions included:

- Exploratory data analysis applied to MRI images.
- Development and evaluation of Machine Learning and Deep Learning models.
- Support in organizing the project workflow and repository structure.
- Integration of components to make the model easier to use through an application.
- Documentation and technical structuring of the repository to make the project more presentable.
